# Деплой базы на Render

Создать облачные ресурсы можно только из вашего аккаунта Render — ниже точный
путь. База поднимается на бесплатном тарифе, апгрейд на платный — одной кнопкой
в дашборде БД, строка подключения при этом не меняется.

## Вариант A — через Blueprint (рекомендую, быстрее)

1. Зайдите на https://dashboard.render.com → **New +** → **Blueprint**.
2. Подключите репозиторий `ekzotik-inc/we_grow` и ветку.
3. Render найдёт [`render.yaml`](../render.yaml) и предложит создать **wegrow-db**
   (free). Нажмите **Apply**.
4. Готово. Строку подключения возьмёте в шаге «Подключение» ниже.

## Вариант B — вручную

1. Dashboard → **New +** → **PostgreSQL**.
2. Заполните:
   - **Name**: `wegrow-db`
   - **Database**: `wegrow`
   - **User**: `wegrow`
   - **Region**: ближайший (напр. Frankfurt)
   - **Plan**: **Free**
3. **Create Database** и подождите статус *Available*.

## Переиспользование существующей free-базы

Render разрешает только **одну** активную free-PostgreSQL на аккаунт. Если она
уже есть — не создавайте новую, а подключитесь к существующей: наши объекты
живут в отдельной схеме `wegrow` и не конфликтуют с тем, что уже лежит в базе
(в `public` или других схемах).

**Internal vs External URL.** В дашборде даётся два адреса. Internal
(`postgresql://user:pass@dpg-xxxx-a/db`, без домена) работает только внутри
сети Render. Для локального запуска нужен **External** — тот же адрес, но хост
с доменом региона, например `dpg-xxxx-a.oregon-postgres.render.com`.

## Подключение бота к базе

1. На странице БД скопируйте **External Database URL** (вид
   `postgresql://wegrow:...@dpg-xxxx.frankfurt-postgres.render.com/wegrow`).
2. Локально в `.env`:
   ```
   DATABASE_URL=<External Database URL с Render>
   ```
   SSL включается автоматически — `bot/db.py` видит хост `render.com` и
   подключается по TLS. Схема применится при первом `python -m bot.main`.
3. Проверка подключения без бота:
   ```bash
   psql "<External Database URL>" -c "\dt"
   ```
   После первого запуска бота увидите таблицы `teams`, `participants` и т.д.

## Апгрейд на платный тариф

Дашборд БД → **Settings** → **Change Plan** → выбрать платный. Данные и строка
подключения сохраняются, ничего в коде и `.env` менять не нужно.

## Важно про бесплатный тариф

- Бесплатная БД Render **живёт 30 дней**, затем её нужно апгрейдить или она
  удаляется — это и есть повод перейти на платный до конца срока.
- Постоянный процесс бота (polling) на Render — это **Background Worker**, он
  платный. Пока БД на free, бота держите локально (`python -m bot.main`), а в
  облако вынесете, раскомментировав блок `services` в `render.yaml` после
  апгрейда.
